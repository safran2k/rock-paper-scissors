# rock-paper-scissors
Rock paper scissors interactive game

An alternative to the classical Rock Paper Scissors game, with identical mechanics but cooler visuals! The element approach (fire, water, grass) was inspired by the Pok√©mon game series! Game plays up to 5 rounds, and displays winner when first person reaches 5 points.
